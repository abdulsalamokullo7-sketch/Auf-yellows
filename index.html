Settings

Data & storage

Your bunches and loans are saved on this device in the browser (localStorage). The app writes the same data twice and checks both copies after each save. Your data survives refresh, closing the tab, and normal phone restarts — it is not sent to any server unless you add one later. It can be lost if you clear site data, uninstall the browser, or use a strict private mode — use Download backup for a safe copy.

Offline & updates

After the first visit, the app shell loads without a network.

Business data is always local — it does not need internet.

Security

PIN is stored as a hash (SHA-256) with a random salt — not as plain text. Unlock persists across refresh in this tab; use Lock app or close the tab to require PIN again.

Recovery (optional)

Save a phone number so you can reset the PIN from the lock screen if you forget it.
